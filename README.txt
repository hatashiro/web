hatashiro/web
=============

Web-related stuff for myself.


Features
--------

- No dependency. No bundling. Only JavaScript modules.
- Literally for myself. Never be production-ready.


How to use
----------

Import each module via jsDelivr.

  import {...} from 'https://cdn.jsdelivr.net/gh/hatashiro/web/...js';

Bundling will never be officially supported.


Versioning
----------

There is no versioning. Backword compatibility will never be ensured. If
consistency is important for you, import from a specific SHA.

  import {...} from 'https://cdn.jsdelivr.net/gh/hatashiro/web@416260/...js';
